Enhance your internal APK!

A library with a collection of tools for your developers and testers. It allows to inspect, report and debug your app from the same screen where it is running. No cable needed, our UI shows over your activities.

Expected release on December 2019

PRE-RELEASE available

Open on Github.com

01.

Repro steps

Track events per sessions. User interactions, network requests, lifecycle events, crashes, ANRs, device events...

02.

Crash handler

View crash details immediately with graphic stacktrace, causing source lines, previous logs and screenshot.

03.

Live info

Get exclusive info about your running app (processes, task, threads, services...), your build (user, host, remote repo, local changes...), your app, the device and his OS.

04.

Advanced inspectors

View layout, standard logcat, source code (original and generated) and storage (db, SharedPrefs and Files).

05.

Reports

Send crashes or issues directly to developers, including a zip with all gathered information.

Easy setup

Install in seconds using Gradle and start using it!

Customizable

Configure our library for your builds, integrate your sources and use our dev helpers.

App respectful

We carefully avoid any interference with your app behaviour, as it could hide you bugs.

Safe Release

Auto disabled by default and noop artifact for minimum apk size increase.
